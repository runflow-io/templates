# Plans

Index of every prototype/feature plan in this repo. Each plan lives in its own folder under one of three stage directories:

```
docs/plans/
├── index.md                       (this file)
├── todo/                          not started — design only
│   └── <plan-name>/
├── progress/                      actively being built or shipped
│   └── <plan-name>/
└── done/                          merged + deployed + verified
    └── <plan-name>/
```

A plan folder typically contains a `design.md` (the spec), and may grow `notes.md`, `deploy.md`, follow-ups, screenshots, etc.

When a plan moves stages, move its folder — don't duplicate. Keep this index in sync.
